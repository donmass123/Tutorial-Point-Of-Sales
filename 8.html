Gambaran Umum

Panduan ini akan membantu Anda mengimplementasikan menu tarik-turun yang nyaman di header aplikasi Anda yang memungkinkan administrator menjalankan perintah pengoptimalan Laravel yang umum tanpa mengakses baris perintah. Fitur ini meningkatkan produktivitas pengembang dan efisiensi pemeliharaan sistem.

Apa yang akan Anda Bangun

Menu tarik-turun header dengan perintah-perintah Laravel berikut ini:

  • Optimalkan Hapus: Menghapus semua cache konfigurasi, rute, tampilan, dan layanan yang dikompilasi
  • Route Clear: Menghapus cache route
  • Config Clear: Menghapus cache konfigurasi
  • View Clear: Menghapus file tampilan yang dikompilasi
  • Cache Clear: Menghapus cache aplikasi
  • System Logs: Akses cepat ke penampil log

Mengatasi masalah

Masalah-masalah Umum:

  • Menu tarik-turun tidak muncul: Periksa apakah sub-kategori diaktifkan di pengaturan bisnis
  • Sub-kategori tidak dimuat: Verifikasi rute AJAX /products/get_sub_categories yang ada
  • Pemfilteran tidak berfungsi: Pastikan kode pemfilteran pengontrol ditambahkan dengan benar
  • Kesalahan JavaScript: Periksa konsol browser untuk mengetahui adanya kesalahan jQuery atau sintaksis
